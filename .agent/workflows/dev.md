# Workflow: Iniciar Desarrollo

Pasos para iniciar el servidor de desarrollo del proyecto Antigravity.

## Prerrequisitos

Asegúrate de tener Node.js >= 18 instalado.

## Pasos

1. Instalar dependencias (si es la primera vez)
// turbo

```bash
npm install
```

2. Iniciar servidor de desarrollo
// turbo

```bash
npm run dev
```

3. El navegador se abrirá automáticamente en `http://localhost:3000`

## Notas

- El servidor tiene hot reload activado
- Los cambios en SCSS se reflejan instantáneamente
- Para detener: `Ctrl + C`
